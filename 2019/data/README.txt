说明

    收集了 上证主板, 深成主板 个股及大盘数据

stoke_dataset.json:      收集了到 2019/2/25 为止的, 上证, 深成 所有个股
    name:           股票名称
    stockCode:      股票编号
    affiliatedExchange: 股票所属交易所(1: 上证, 2: 深成)


stock_index.json:       主板信息
    name:           主板名称
    stockCode:      主板编号
    affiliatedExchange: 没有什么用, ... 主要是用于爬取数据时的一个标示


details: 存放上述个股自上市以来所有交易日详细数据, 文件名称是股票编号
    第一列: 日期
    第二列: 开盘价(开盘价不一定等于上一个交易日收盘价, 开盘之前有竞价阶段, 开盘价可能大于/小于/等于上一个交易日的收盘价)
        : 收盘价
        : 盘中最高价
        : 盘中最低价
        : 成交量(单位: 手(1手 = 100股))
        : 成交金额(单位: 人民币)
        : 振幅
        : 换手率


index_details: 存放主板自开板以来所有交易日的详细数据, 文件编号是主板编号
    第一列: 日期
    第二列: 开盘指数
        : 收盘指数
        : 盘中最高指数
        : 盘中最低指数
        : 成交量(单位: 手)
        : 成交金额(单位: 人民币)
        : 振幅
        : 换手

    其实和details内的个股参数意思基本一致
